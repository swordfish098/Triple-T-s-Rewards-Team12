Instructions

Upload a pipe-delimited text file to create multiple users at once.

Important: Organizations must be created before creating sponsors or drivers for that organization.

Format for the file:

  • O|Organization Name - Create an organization
  • S|Organization Name|First Name|Last Name|Email Address - Create a sponsor user for an existing organization
  • D|Organization Name|First Name|Last Name|Email Address - Create a driver user for an existing organization

Note: Driver license numbers will be added separately after bulk loading.

Example (in proper order):

O|Triple T's Trucking
S|Triple T's Trucking|Jane|Smith|jane.smith@triplets.com
D|Triple T's Trucking|John|Doe|john.doe@gmail.com
